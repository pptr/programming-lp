# NK Works 問い合わせシステム 要件定義

## 1. 概要
Webサイトからの問い合わせを受け付け、管理者（`info@nkworks.info`）へメール通知する。DB保存・管理画面は実装しない。

## 2. 入力項目
- 氏名（必須、1～100文字）
- メールアドレス（必須、形式チェック）
- 電話番号（任意）
- お問い合わせ種別（必須）
- お問い合わせ内容（必須、10～2,000文字）
- プライバシーポリシー同意（必須）

## 3. 画面遷移
入力 → 確認 → メール送信 → 完了

## 4. セキュリティ・スパム対策
- CSRF対策
- XSS対策
- SMTPヘッダーインジェクション対策
- 二重送信防止
- HTTPS必須
- Honeypot
- 送信間隔制限
- Cloudflare Turnstile（将来導入可能）

## 5. 技術要件
- PHP 8.2
- PHPMailer
- SMTP認証
- UTF-8
