# NK Works 問い合わせシステム 要件定義

## 1. 概要

### 目的

Webサイトからの問い合わせを受け付け、管理者（`info@nkworks.info`）へメール通知を行う。

### 対象システム

-   NK Works Webサイト
-   PHP 8.2
-   Apache
-   独自ドメイン：nkworks.info

------------------------------------------------------------------------

# 2. システム構成

    利用者
        ↓
    お問い合わせフォーム
        ↓
    入力チェック
        ↓
    確認画面
        ↓
    送信
        ├── 管理者メール送信
        └── 自動返信メール送信
        ↓
    完了画面

※ DB保存・管理画面は実装しない。

------------------------------------------------------------------------

# 3. 機能要件

## 3.1 入力項目

  項目                       必須   備考
  -------------------------- ------ ----------------------
  氏名                       〇     1～100文字
  メールアドレス             〇     形式チェック
  電話番号                   任意   半角数字・ハイフン可
  お問い合わせ種別           〇     プルダウン
  お問い合わせ内容           〇     10～2,000文字
  プライバシーポリシー同意   〇     チェックボックス

### お問い合わせ種別

-   レッスン内容について
-   受講方法について
-   料金について
-   予約について
-   学習相談
-   その他

------------------------------------------------------------------------

## 3.2 入力チェック

-   必須入力
-   メールアドレス形式
-   最大文字数
-   HTMLタグ除去
-   前後空白除去
-   改行保持

------------------------------------------------------------------------

## 3.3 画面遷移

    入力：問い合わせ内容入力フォームの表示
     │ ↑①
     ├─┘
     ↓②
    確認：入力内容を表示
     │ ↑③
     ├─┘
     ↓④
    完了：送信完了のお知らせ画面

    ①：送信ボタン押下後、入力内容のエラーがあった場合（例：未入力、メールアドレスのフォーマットエラー等）に遷移
    　　※エラー内容の表示
    ②：送信ボタン押下後、入力内容に問題がない場合に遷移
    ③：確認ボタン押下後、エラー発生時に遷移
    　　※エラーメッセージを表示し、直接メール問い合わせを誘導
    ④：確認ボタン押下後、正常終了時に遷移

------------------------------------------------------------------------

## 3.4 メール送信

### 管理者向け

宛先
    info@nkworks.info

件名
    【お問い合わせ】<氏名> 様

内容
-   氏名
-   メールアドレス
-   電話番号
-   お問い合わせ種別
-   お問い合わせ内容
-   送信日時

------------------------------------------------------------------------

### 自動返信

宛先
    <入力メールアドレス>
件名
    【NK Works】お問い合わせありがとうございます

内容
-   受付完了
-   返信までの目安
-   問い合わせ内容

本文
[氏名] 様

この度は、お問い合わせありがとうございました。
以下の内容でお問い合わせを受け付けいたしました。


【受付日時】[送信日時]
【お名前】[お客様のお名前] 様
【メールアドレス】[メールアドレス]
【お問い合わせ種別】[問い合わせ種別]
【お問い合わせ内容】
[本文]


内容を確認次第、担当者より3営業日以内にご連絡させていただきます。
今しばらくお待ちいただけますようお願い申し上げます。

※本メールはシステムによる自動送信となります。

# 8. ログ

以下を/apps/contant/logファイルへ出力する。

## 正常ログ

ログレベル=1: info
- メール送信成功: mail sent successfully.

## 異常ログ

ログレベル=2: error
- メール送信失敗: sending mail failed.
- その他例外発生: 任意のメッセージ

## 形式
[yyyy-mm-dd hh:mm:ss][level] message

例）[2026-05-12 18:35:10][info] mail sent successfully.
