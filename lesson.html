レッスン詳細

一人ひとりの経験や目標に合わせて、 基礎学習から作品制作、学校課題、コードレビューまで柔軟に対応します。

レッスンの特徴

現在の理解度や学習目的を確認し、 NK Works独自教材と受講者様の取り組みたい題材を組み合わせて、 一人ひとりに合わせたレッスンを行います。

完全マンツーマン

講師が一人ひとりの理解度を確認しながら進めます。 分からない部分をそのままにせず、 質問や確認を挟みながら自分のペースで学習できます。

オーダーメイドの学習内容

プログラミング未経験者の基礎学習から、 学校課題、作品制作、コードレビュー、就職・転職に向けた学習まで、 目的に合わせて内容を調整します。

作りながら身につける

文法を覚えるだけで終わらず、 アプリやゲームなどの制作を通して、 プログラムを考え、組み立て、完成させる力を身につけます。

学べる内容

目的や経験に応じて、基礎から実践まで段階的に学習します。 以下は対応内容の一例です。

プログラミング基礎

  • 開発環境の構築と基本操作
  • 変数・型・演算子
  • 条件分岐・繰り返し
  • 関数・配列
  • エラーの読み方・調べ方

設計・アルゴリズム

  • クラスとオブジェクト
  • オブジェクト指向設計
  • データ構造とアルゴリズム
  • 処理の分割と責務
  • 読みやすく修正しやすいコード

制作・実践

  • ミニゲーム制作
  • 簡単なアプリ制作
  • 既存プログラムへの機能追加
  • コードレビューと改善
  • 制作物の完成までのサポート

目的別のレッスン例

BEGINNER

初めてプログラミングを学ぶ

パソコンの基本操作や開発環境の準備から始め、 変数、条件分岐、繰り返し、関数などを順番に学びます。

基礎を身につけた後は、 簡単なゲームやアプリの制作に取り組みます。

STUDENT

学校の授業・課題を理解する

授業で分からなかった内容や課題のコードを確認し、 どこでつまずいているのかを一緒に整理します。

答えだけを示すのではなく、 自分で考えて進められるようになることを目指します。

CREATION

ゲームやアプリを制作する

作りたいものを相談し、 必要な機能を整理して、段階的に実装します。

設計、実装、テスト、修正を経験しながら、 一つのプログラムを完成させます。

REVIEW

自作コードを改善する

現在制作しているプログラムを題材に、 コードレビューや設計相談を行います。

読みやすさ、保守性、処理の分割など、 実際の開発でも役立つ観点をお伝えします。

CAREER

就職・転職に向けて学ぶ

基本文法やアルゴリズムを整理しながら、 制作物の作成やコードの説明練習などを行います。

現在の経験と目標に応じて、 必要な学習内容をご提案します。

CUSTOM

その他の内容を相談する

掲載されていない内容についても、 講師が対応可能な範囲で柔軟にレッスンを組み立てます。

受講前に確認したい場合は、無料相談をご利用ください。

通常レッスンの進め方

継続レッスンでは、目標に向けて講義・演習・制作を組み合わせながら進めます。 毎回必ず同じ構成ではなく、当日の内容に応じて時間配分を調整します。

1

前回内容・進捗の確認

前回までに学んだ内容や、 レッスン外で取り組んだ内容を確認します。

分からなかった点やエラーがあれば、 ここで一緒に整理します。

2

講義・演習

新しく学ぶ内容を説明し、 短いコードや練習問題を通じて理解を深めます。

講師の説明だけで終わらず、 実際に手を動かしながら進めます。

3

制作・振り返り

学んだ内容を制作物や実際の課題へ取り入れます。

最後に今回の内容を振り返り、 次回までに取り組む内容を確認します。

対応言語・対象

主な対応言語

  • C++
  • C
  • Java
  • PHP

※その他の言語や技術については、受講前にお問い合わせください。

対象

  • 中学生以上
  • 学生・社会人不問
  • 未経験・初心者歓迎
  • 経験者のコードレビューにも対応

対応できる相談

  • 学習内容が決まっていない
  • 自分に合う言語を知りたい
  • 制作物のテーマを相談したい
  • 現在のコードを見てほしい

教材について

基礎学習ではNK Works独自教材を使用します。 学校の教材、自作プログラム、現在取り組んでいる課題を持ち込んでいただくことも可能です。

市販教材の購入をお願いする場合は、 必ず事前にご相談します。
